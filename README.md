# OpenClaw Security Skills

A small skill pack for OpenClaw focused on host hardening, agent security, prompt-injection review, approval surfaces, sandboxing, and trust-boundary analysis.

## Included skills

- `healthcheck`
  - Host hardening and OpenClaw deployment security workflow
- `agent-security`
  - Runtime, approval, tooling, prompt-injection, and trust-boundary hardening

## Layout

- `skills/healthcheck/`
- `skills/agent-security/`
- `dist/` for packaged archives

## Notes

These skills are designed for OpenClaw-style AgentSkills and include references and helper scripts.
